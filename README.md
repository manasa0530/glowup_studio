## 🌟 GlowUp Studio – Responsive Fashion Website

GlowUp Studio is a modern and fully responsive fashion website built using HTML, CSS, and Bootstrap.  
This project focuses on clean UI design, structured layout, and smooth user experience across all devices.

### 🔗 Live Demo
👉 https://glowupstudio1.netlify.app/


###  Project Overview

This is a static front-end project developed as part of my learning journey in web development.  
The website represents a fashion brand concept and demonstrates how real-world fashion websites structure their layout and design.

The main goal of this project was to strengthen my front-end fundamentals and improve responsive design skills.

### Features

- Clean and modern homepage design  
- Fashion product showcase using card layouts  
- Fully responsive (Mobile, Tablet, Desktop)  
- Bootstrap-based consistent UI components  
- Structured sections with proper spacing and alignment  
- User-friendly navigation  

### Technologies Used

- HTML5  
- CSS3  
- Bootstrap 5
  
### Learning Outcomes

Through this project, I improved my understanding of:
- Responsive web design principles  
- Layout structuring and content organization  
- CSS styling and spacing techniques  
- Bootstrap grid system and components  
- Writing clean and structured front-end code
